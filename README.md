Savvy

Сучасний веб-додаток для онлайн-шопінгу, побудований на стеку React, Node.js, Express та PostgreSQL з інтеграцією зовнішнього API DummyJSON.

ОСНОВНІ МОЖЛИВОСТІ

* Каталог товарів: Динамічне завантаження товарів через DummyJSON API.
* Детальна сторінка товару: Перегляд зображень, описів, характеристик та автоматичний перерахунок ціни в гривнях.
* Система відгуків: Можливість залишати коментарі та ставити оцінки товарам.
* Автоматична модерація: Вбудована система фільтрації нецензурної лексики (українська та англійська мови) на фронтенді та бекенді.
* Список обраного: Збереження улюблених товарів у базі даних з прив’язкою до профілю користувача.
* Кошик: Функціонал додавання товарів та керування покупками.
* Автентифікація: Захист даних користувача та сесій за допомогою JWT (JSON Web Tokens).

ТЕХНОЛОГІЧНИЙ СТЕК

Фронтенд:

* React (TypeScript)
* Tailwind CSS для стилізації
* React Router DOM для навігації
* Бібліотека bad-words для модерації

Бекенд:

* Node.js та Express
* PostgreSQL (База даних)
* JWT для безпеки
* Спеціальні скрипти для обробки та збереження даних користувачів

БЕЗПЕКА ТА МОДЕРАЦІЯ

1. Двохрівнева перевірка: Коментарі проходять валідацію спочатку на клієнтській стороні (для миттєвого відгуку), а потім на серверній (для забезпечення чистоти бази даних).
2. Захист маршрутів: Додавання відгуків та робота з обраним доступні лише авторизованим користувачам.
3. Модерація: Використання бібліотеки bad-words для відслідковування неприйнятної лексики, сприяючи комфортним умовам веденню торгівлі.

ЯК ЗАПУСТИТИ ПРОЕКТ

1. Встановіть залежності в папках клієнта та сервера за допомогою команди npm install.
2. Налаштуйте змінні середовища у файлі .env (підключення до БД, секретний ключ JWT).
3. Запустіть сервер (npm run dev) та фронтенд (npm start).
